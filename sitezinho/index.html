O que é Programação?

Programação é o processo de escrever instruções para computadores seguirem. Essas instruções, chamadas de códigos, são escritas em linguagens de programação que os humanos podem aprender e os computadores podem entender.

A programação é importante porque nos permite criar softwares, aplicativos, sites e sistemas que resolvem problemas, automatizam tarefas e melhoram nossas vidas.

// Exemplo simples de código
console.log("Olá, Mundo!");

Variáveis e Tipos de Dados

Variáveis são como caixas que armazenam informações. Cada variável tem um nome e um valor.

Tipos de dados básicos:

  • Inteiros: Números inteiros (ex: 5, -10, 1000)
  • Decimais: Números com casas decimais (ex: 3.14, -0.5, 2.0)
  • Textos (Strings): Sequências de caracteres (ex: "Olá", "Python", "123")
  • Booleanos: Valores verdadeiro ou falso (true/false)
// Declarando variáveis
let idade = 23;               // Inteiro
let altura = 1.75;            // Decimal
let nome = "Fabrício";           // String
let estudante = true;         // Booleano

Estruturas Condicionais

As estruturas condicionais permitem que o programa tome decisões com base em condições.

If e Else

O comando if verifica se uma condição é verdadeira. Se for, executa um bloco de código. O else executa um bloco alternativo se a condição for falsa.

let nota = 85;

if (nota >= 70) {
    console.log("Aprovado!");
} else {
    console.log("Reprovado!");
}

Laços de Repetição

Laços de repetição permitem executar um bloco de código várias vezes.

For

O loop for é usado quando sabemos quantas vezes queremos repetir o código.

// Imprime números de 1 a 5
for (let i = 1; i <= 5; i++) {
    console.log(i);
}

While

O loop while repete o código enquanto uma condição for verdadeira.

// Contagem regressiva
let contador = 5;
while (contador > 0) {
    console.log(contador);
    contador--;
}
console.log("Feliz Ano Novo!");

Funções

Funções são blocos de código reutilizáveis que realizam uma tarefa específica.

Elas ajudam a organizar o código, evitar repetição e facilitar a manutenção.

// Definindo uma função
function cumprimentar(nome) {
    return "Olá, " + nome + "!";
}

// Usando a função
let mensagem = cumprimentar("Carlos");
console.log(mensagem); // "Olá, Carlos!"
